ffmpeg logo overlay in PHP
==========================

Add a watermark to a video in PHP

## Usage

Docker required.

1. `make start`
2. `make deps`
3. `make sample`
4. `make magic`
5. `make stop`

See the result available in the `app/data/result.mp4` file.
